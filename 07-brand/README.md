# 07 — NHẬN DIỆN THƯƠNG HIỆU

Tài sản nhận diện dùng cho tài liệu, hồ sơ, thuyết trình.

## Tài liệu cần có
- [ ] Logo — đủ định dạng: `.png` (nền trong), `.svg`, `.ai`, bản đơn sắc
- [ ] Bảng màu thương hiệu (mã HEX/CMYK)
- [ ] Bộ font chính / phụ
- [ ] Mẫu letterhead / văn bản có header-footer công ty
- [ ] Con dấu / chữ ký số hóa (dùng cho bản mềm)
- [ ] Template slide thuyết trình

## Quy ước đặt tên
`logo_<biến-thể>.<ext>` — vd `logo_full-color.svg`, `logo_mono-white.png`
