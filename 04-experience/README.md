# 04 — KINH NGHIỆM

Hồ sơ chứng minh kinh nghiệm thực hiện hợp đồng tương tự — tiêu chí kỹ thuật quan trọng.

## Tài liệu cần có
- [ ] Danh mục hợp đồng tương tự đã thực hiện (bảng tổng hợp)
- [ ] Bản sao hợp đồng đã ký
- [ ] Biên bản nghiệm thu / thanh lý hợp đồng
- [ ] Thư cảm ơn / xác nhận của chủ đầu tư
- [ ] Hình ảnh / sản phẩm dự án tiêu biểu (portfolio)

## Quy ước đặt tên
Mỗi dự án một thư mục con: `<năm>_<tên-dự-án>/` chứa hợp đồng + nghiệm thu + ảnh.
Bảng tổng hợp: `danhmuc-hopdong-tuongtu.xlsx`
