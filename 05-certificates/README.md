# 05 — CHỨNG CHỈ & CHỨNG NHẬN

Chứng chỉ năng lực, hệ thống quản lý, giải thưởng — tăng điểm đánh giá.

## Tài liệu cần có
- [ ] Chứng nhận hệ thống quản lý (ISO 9001, ISO 27001, ...)
- [ ] Chứng chỉ năng lực hoạt động theo ngành
- [ ] Chứng nhận đối tác / hãng (partner, reseller, ...)
- [ ] Giải thưởng, bằng khen
- [ ] Chứng nhận thành viên hiệp hội ngành nghề

## Quy ước đặt tên
